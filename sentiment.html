Project Overview

This project uses sentiment analysis to study the correlation between Reddit's Wall Street Bets subreddit posts and the performance of top S&P 500 companies. We employed natural language processing tools to analyze sentiment data.

Key Achievements

  • Collected data from Reddit and Yahoo Finance APIs for companies like Tesla and Apple.
  • Performed sentiment analysis using VADER, analyzing the impact of social media sentiment on stock performance.
  • Built multiple linear regression models to predict stock price movements based on sentiment data.

Tools and Technologies Used

  • Python, Pandas
  • Reddit API, Yahoo Finance API
  • VADER Sentiment Analysis
